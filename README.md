# Minha API

Este projeto será uma API RESTful que fornecerá funcionalidades para gerenciamento de recursos (exemplo: usuários, produtos, pedidos).

## Objetivos
- Criar endpoints para operações CRUD.
- Implementar autenticação e autorização.
- Documentar a API com OpenAPI/Swagger.
- Garantir qualidade com testes automatizados.

## Tecnologias previstas
- Linguagem: Node.js
- Framework: Express
- Banco de dados: PostgreSQL

## Estrutura inicial
1. Configuração do servidor.
2. Definição dos modelos de dados.
3. Implementação dos primeiros endpoints.
